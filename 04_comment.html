註解範例

這是註解範例與介紹。

HTML 單行註解，快速鍵Ctrl+C

<!-- Lorem ipsum dolor sit amet. -->

HTML 多行註解，快速鍵Ctrl+C

<!-- Lorem ipsum dolor sit amet.
Lorem ipsum dolor sit amet. -->

CSS 單行註解，快速鍵Ctrl+C

/* Lorem ipsum dolor sit amet. */

CSS 多行註解，無快速鍵Ctrl+C

/* Lorem ipsum dolor sit amet.
Lorem ipsum dolor sit amet. */

C C++ C# JavaScript Java PHP 單行註解，快速鍵Ctrl+C

// Lorem ipsum dolor sit amet.

C C++ C# JavaScript Java PHP 多行註解，快速鍵Ctrl+C

/* Lorem ipsum dolor sit amet.
Lorem ipsum dolor sit amet. */

Python 單行註解，快速鍵Ctrl+C

# Lorem ipsum dolor sit amet.

Python 多行註解，快速鍵Ctrl+C

"""
Lorem ipsum dolor sit amet.
Lorem ipsum dolor sit amet.
"""
